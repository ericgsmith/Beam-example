Alerts can be used to display important messages.

Information alert
Warning alert
Error alert
Success alert
<div class="alert alert--info">Information alert</div>

<div class="alert alert--warning">Warning alert</div>

<div class="alert alert--error">Error alert</div>

<div class="alert alert--success">Success alert</div>

Buttons

Buttons are typically applied to a or button elements.

A link. 
<a href="#" class="button">A link.</a>
<button class="button">A button</button>

Button sizes

Buttons have 3 sizes: small, medium (default), and large. Buttons also have an additional modifier class of full width.

Button Modifier Class
button--large
button
button--small
button--fullwidth

Panels

Panels can be used to display content within the site.

Stuff inside a panel

 
<div class="panel">
  <div class="panel__content">
    <p>Stuff inside a panel</p>
   </div>
</div>

Panels can be with titles.

Panel title
